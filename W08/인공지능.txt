[인공지능]

딥러닝 프레임워크 종류

1. 텐서플로우(TensorFlow)

- 데이터 플로우 그래프(Data Flow Graph) 사용

- 수학 계산식과 데이터의 흐름을 노드와 엣지를 사용한 방향성 그래프로 표현

- 이미지 인식, 기계 번역 등의 신경망 학습에 이용

- 대규모 예측 모델 구성에 뛰어남

 

2. 케라스(Keras)

- 케라스에서 제공하는 시퀀스 모델로 원하는 레이어를 쉽게 순차적으로 쌓을 수 있음

- 일반 사용 사례에 최적화된 인터페이스 제공

- 사용자 오류에 대한 명확하고 실용적인 피드백 제공

- 모듈화의 한계로 복잡한 것에 대한 구현 범위 좁은 편

 

3. 파이토치(Pytorch)

- 절차 간단

- 그래프가 동적으로 변화

- Define by run 방식으로 깔끔하고 직관적인 코드 작성 가능

- 학습 속도 빠름

- 메모리에서 연산하며, 신경망 사이즈를 최적으로 바꾸면서 동작 가능
